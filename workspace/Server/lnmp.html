# 添加、删除虚拟主机使用教程

# 添加
# 如果输入有错误需要删除时，可以按住Ctrl再按Backspace键进行删除。
lnmp vhost add 

# 列表
lnmp vhost list

# 删除网站会先列出当前已有虚拟主机，按提示输入要删除的虚拟主机域名 回车确认。
# 这里只是删除虚拟主机配置文件，网站文件并不会删除需要自己删除。
lnmp vhost del

# 安装node

# 版本升级
npm i -g n
rm -rf /usr/bin/node
n stable  # 最稳定的版本
ln -s /usr/local/bin/node /usr/bin/node
node -v  # v12.14.0
